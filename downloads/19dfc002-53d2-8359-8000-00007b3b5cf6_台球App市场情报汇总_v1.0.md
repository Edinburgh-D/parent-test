# 台球App市场情报汇总 · 2025年5月

## 核心结论

**中国没有"纯台球社交平台"，所有产品都是"通用搭子平台"或"台球资讯站"的杂交形态。**
**最大的垂直台球App（台球会）只有约球+计分+资讯三合一，没有社交、没有电商、没有门店SaaS。**
**这是明确的市场空白。**

**关键发现**：这些平台用户量数字水分极大（6800万注册用户≠台球活跃用户），且台球功能浅到只有"发布台球需求"和"搜索台球tag"。

### B类：台球垂类App（真正和台球有关）

| App | 功能 | 状态 | 评价 |
|-----|------|------|------|
| **台球会** | 资讯+视频+约球+计分器 | 运营中 | 最接近你想象的产品，但UI老旧、无社交链、无电商 |
| **嘿吧** | 台球交友+圈子 | 运营中 | 社交功能有但无计分/门店/商城 |
| **台球帝** | 资讯+赛事 | 运营中 | 纯内容站，无约球 |
| **准梦台球预约** | 球房预订+群聊+私聊 | 小米商店上架 | 有预订和社交，但无计分/商城/比赛记录 |
| **台球计分器** | 纯计分工具 | 独立App | 支持比洞/比杆，但无社交、无网络功能 |
| **博客台球** | 商品展示+购物 | 疑似停运 | 纯电商，无社区 |
| **台球斯诺克** | 新闻资讯 | 运营中 | 纯内容 |

**关键发现**：
- 没有一款App同时覆盖"约球+计分+门店+商城+社交"五大模块
- 台球会是最强的对手，但产品形态停留在2018年
- 所有计分工具都是离线单机版
- 没有门店SaaS系统（球房还在用微信群+现金）

### C类：台球游戏App（伪需求，不算竞品）

- 逢友、爽搭等——本质是台球对战手游，不是约球平台

## 二、竞争强度评估

| 维度 | 评分 | 说明 |
|------|------|------|
| 垂直领域竞争 | ⭐⭐☆☆☆ | 几乎没有专业级产品 |
| 通用平台替代风险 | ⭐⭐⭐☆☆ | 咕哇/丛丛可临时约台球 |
| 技术门槛 | ⭐⭐⭐☆☆ | WebSocket实时计分是难点 |
| 用户获取成本 | ⭐⭐⭐⭐☆ | 台球爱好者集中在线下 |
| 变现难度 | ⭐⭐⭐☆☆ | 门店SaaS+电商+会员三层可跑通 |

**结论：蓝海市场，但用户获取依赖线下地推。**

## 四、你项目的差异化空间

**5个"没有竞品在做"的核心功能**：

1. **实时比赛计分同步**——两人同屏计分，WebSocket推送，历史回放
2. **台球门店SaaS**——球房入驻、空台状态、预订系统、会员管理
3. **装备垂直电商**——台球杆、皮头、巧粉、手套，专业评测+社区种草
4. **个人技术档案**——胜率、清台率、出杆数据（未来接AI分析）
5. **赛事系统**——从约球到自发小型比赛到城市联赛
